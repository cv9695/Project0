Basics of the Web & HTML

HTTP & HTML

HTTP stands for Hypertext Transfer Protocol. HTTP is the primary protocol used for Internet communication between the computer's browser and the Web server.

HTML stands for Hypertext Markup Language. HTML is the main type of document language on the web. HTML can include text content, as well as links to other documents, photos, and videos.

HTML Tags & Elements

HTML tags are the heart of HTML. Tags are used to design the text portion of the document, as well as add more detail, such as photos, videos, and links to other documents.

Tag Example #1: <a href="www.udacity.com">UDACITY</a>

This example above shows a HTML anchor tag to create a hyperlink on the word UDACITY.

Tag Example #2: <em>HELLO!</em>

This example shows the <em> tag, which place its contents in italics......HELLO!

Inline & Block Elements

Inline Elements work within the same "text box", whereas Block elements create a new invisible "text box".
